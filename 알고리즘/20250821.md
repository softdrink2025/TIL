### 너비우선탐색 BFS(Breadth First Search)
- 탐색 시작점의 인접한 정점들을 모두 차례로 방문한 후에, 방문했던 정점을 시작점으로 하여 다시 인접한 정점들을 차례로 방문하는 방식

# 너비우선탐색
def BFS(s, V):
    # 초기화 
    visited = [0] * (V + 1)  # visited 생성
    q = []       # 큐 생성
    q.append(s)  # 시작점 인큐
    visited[s] = 1   # 시작점 인큐 표시

    # 반복
    while q:  # 탐색할 정점이 남아 있으면
        t = q.pop(0)  # 디큐
        print(t)      # visit(), 방문 정점 출력
        for w in adj_list[t]:
            if visited[w] == 0:
                q.append(w)
                visited[w] = visited[t] + 1
